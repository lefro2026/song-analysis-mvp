# Song Analysis MVP

アカペラ音声をアップロードすると、音高・高音傾向・音量推移を簡易解析できるWebアプリです。  
スマホからでも利用できます。

---

## 主な機能

- 音声ファイルアップロード
- wav / m4a 対応
- iPhoneボイスメモ対応
- Android録音ファイル対応（一部機種）
- 前後の静音区間を自動除去
- 最高音の表示
- 平均音高の表示
- 高音維持時間
- 高音割合
- 音量推移グラフ
- 音高推移グラフ
- 簡易コメント表示

---

## 対応ファイル形式

- `.wav`
- `.m4a`

※ `.m4a` は内部で変換して解析します。

---

## 分析可能な長さ

- 0秒 ～ 5分

※ 5分を超える音声は解析できません。

---

## 推奨する音声

- アカペラ録音
- ボーカルがはっきり聞こえる音源
- 雑音が少ない環境

---

## iPhoneからアップロードする方法

1. **ボイスメモ**で録音する  
2. 対象の録音を開く  
3. **共有** を押す  
4. **ファイルに保存** を選ぶ  
5. このアプリでアップロードする

---

## Androidからアップロードする方法

1. 端末の録音アプリで録音する  
2. 録音一覧から対象ファイルを開く  
3. **共有** を押す  
4. ファイル保存、またはこのアプリへアップロードする

### 例

- Google Pixel → Recorder
- Galaxy → Voice Recorder

※ Androidは機種により名称・手順が異なります。
